PyFocus
==================================

A Python script for identifying the active AF points in Pentax RAW files.

Requirements
----------------------------------

* python3
* `sys`, `io`
* [`exiftool`](https://smarnach.github.io/pyexiftool/)
* [`rawkit`](https://rawkit.readthedocs.io/en/latest/)
* [`pillow`](http://python-pillow.org)
